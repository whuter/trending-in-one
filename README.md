## 使用

如想获取**2020-11-29**当天的知乎热搜，只需访问[**https://cdn.jsdelivr.net/gh/hu-qi/trending-in-one/raw/zhihu-search/2020-11-29.json**](https://cdn.jsdelivr.net/gh/hu-qi/trending-in-one/raw/zhihu-search/2020-11-29.json) 即可；
